# Shwe Casino 99

Shwe Casino 99 - 緬甸最佳線上賭場平台

## 技術棧

- Next.js 14
- TypeScript
- Tailwind CSS
- React 18

## 本地開發

```bash
npm install
npm run dev
```

訪問 [http://localhost:3000](http://localhost:3000)

## 構建

```bash
npm run build
npm start
```

## 部署到 Vercel

1. 將代碼推送到 GitHub
2. 在 [Vercel](https://vercel.com) 導入項目
3. Vercel 會自動檢測 Next.js 項目並部署

## 項目結構

- `app/` - Next.js App Router 目錄
  - `page.tsx` - 主頁面
  - `layout.tsx` - 根布局
  - `globals.css` - 全局樣式
  - `sitemap.ts` - 網站地圖
  - `robots.ts` - 機器人配置
