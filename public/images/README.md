## 圖片要求：

- 格式：JPG 或 PNG
- 建議尺寸：

  - 遊戲卡片: 400x400px 或更大
- 文件大小：盡量壓縮以提升載入速度

## 使用方式：

圖片放置後，代碼會自動使用這些圖片。如果某張圖片不存在，會顯示默認的 emoji 圖標。
